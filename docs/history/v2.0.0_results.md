## Not yet accepted

This branch is not v2.0 GA. The following release gates remain open:

- Docker is unavailable on the verification host, so PostgreSQL/Redis/Celery/MinIO/Qdrant Compose, forced-RLS integration, worker-kill, Redis restart, duplicate broker delivery, PITR, and rolling-upgrade drills have not run.
- Team identity/workspace persistence is not complete. Team Analysis/Index has a scoped
  handler; all other unmigrated Team handlers fail closed with a stable unavailable code.
- Full `/api/v2` coverage, unified resource frontend, SDK/CLI breadth, Local-to-Team
  executor, Team backup/restore and the two complete E2E release journeys remain phased work.
- Supply-chain scanners and immutable production image digests have not been run.
- `ALIGNMENT_BENCHMARK_PENDING` remains open with 0 Dev and 0 Locked human-Gold pairs and blocks RC/GA.

No production-readiness or GA claim may cite this document until every open gate is replaced by reproducible evidence.
